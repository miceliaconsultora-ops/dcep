# PRD: Centro de Entrenamiento de Padel Diego Cipriano ## Overview A high-performance web app for padel players to track their technical and physical progress. ## Target Audience Padel players at the Diego Cipriano Training Center. ## Key Features 1. **Athlete Dashboard**: Personal profile (Photo, Name, Age, Weight, Height, Position). 2. **Technical Performance Sheet**: Quantitative (1-10) and qualitative evaluation for core strokes: - Drive & Backhand - Wall exits (Drive/Backhand) - Volleys - Bandeja - Smash (Forward/x3) 3. **Spider Chart**: Visual comparison of all technical skills. 4. **Evolution History**: Filterable history to track progress over time. 5. **Booking Calendar**: Schedule training sessions. ## Design Language - **Style**: Modern, athletic, clean. - **Theme**: Dark Mode with fluo green/yellow accents. - **Components**: Cards, progress bars, radar charts. - **Navigation**: Mobile-friendly bottom navigation.
